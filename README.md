# Errr0rr404/homebrew-tap

Homebrew formulae maintained by [@Errr0rr404](https://github.com/Errr0rr404).

## Install

```bash
brew install errr0rr404/tap/cli-connect
```

Or:

```bash
brew tap errr0rr404/tap
brew install cli-connect
```

## What's here

- **cli-connect** — control your Mac or Windows terminal from your iPhone.
  ([source](https://github.com/Errr0rr404/cli-connect) · [npm](https://www.npmjs.com/package/@worldofz/cli-connect))
